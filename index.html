Console Log

To open the console log, right click and click on inspect. Then, click "console"

According to the website "Stack overflow", console. log specifically is a method for developers to write code to inconspicuously inform the developers what the code is doing. It can be used to alert you that there's an issue, but shouldn't take the place of an interactive debugger when it comes time to debug the code.
